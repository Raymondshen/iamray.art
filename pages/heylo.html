Summary

At Heylo, I teamed up with the founder to tackle challenges discovered through user interviews and research. This involved analyzing competitor products, emphasizing features, UX, and grasping customer problems and needs. Initially, Heylo's product lacked a proper user interface and experience design, hindering a smooth and enjoyable user journey.

app store

Revamped the app store screenshots, revitalizing Heylo's brand, value, and alignment with the target audience. The old designs appeared outdated, featuring mismatched elements that failed to effectively communicate Heylo's identity and purpose.

App Redesign

Revitalized Heylo's core and subpages for enhanced productivity and usability. The previous design had issues with inconsistent button sizes, discoverability, and lacked proper branding color and design principles for a seamless user journey.

Onboarding

Streamlined the group creation onboarding for Heylo, fixing inconsistencies in branding, buttons, input fields, and flow. The original design lacked user resonance, and the redesign aimed for uniformity and consistency with Heylo's branding colors. New features were seamlessly integrated into the onboarding flow to enhance the overall group creation experience.

Chatroom

Revamped the chatroom experience to enhance user engagement, visuals, and productivity, focusing on improving overall user experience. Addressed issues like inconsistent button sizes, colors, and grid design principles for better clarity and navigation.

Events

Enhanced the event UI, emphasizing crucial components for improved user experience and discoverability. Streamlined the flow, reducing event creation time. Worked on new features and enhanced specific areas for clearer user communication.
